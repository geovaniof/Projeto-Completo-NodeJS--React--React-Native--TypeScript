Call Stack
------------------
|                |
|                |
|                |
|                |
|                |
|                |
|                |
|função3()       |
|função2()       |
|função1()       |
_________________|

LIFO - Last IN, First-Out -> Último a entrar, primeiro a sair.
Logo, primeiro será executado a função3(), depois a função2() e por último a função1().
